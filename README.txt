
Este repositório contém estudos a partir do livro 'Entendendo Algoritmos',
de autoria de Aditya Y. Bhargava. Especificamente, escrevem-se os códigos 
dos algoritmos apresentados a cada capítulo, a título de exercitar o aprendizado.
A linguagem de programação é a mesma utilizada no livro pelo autor: Python.

Bibliografia

BHARGAVA, Aditya Y.. Entendendo algoritmos: um guia ilustrado para programadores e 
outros curiosos. Novatec Editora Ltda., 1 ed., 2017.
